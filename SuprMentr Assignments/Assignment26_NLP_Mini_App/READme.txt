Assignment Name: NLP Mini App
Description: Build a simple Chatbot

---

1. Introduction

Natural Language Processing (NLP) enables machines to understand human language. Chatbots are a common application of NLP used in customer support and virtual assistants.

---

2. Objective

To build a simple chatbot that responds to user input using basic NLP techniques.

---

3. Approach

A rule-based approach is used:

* User input is converted to lowercase
* Keywords are matched with predefined responses
* Appropriate response is returned

---

4. Working

The chatbot continuously takes user input and checks for keywords like "hello", "bye", etc. Based on the input, it generates a response.

Example:

User: Hello
Bot: Hello! How can I help you?

User: What is your name?
Bot: I am a simple NLP chatbot.

---

5. Conclusion

This chatbot demonstrates basic NLP concepts and how machines can interact with users using simple logic.
