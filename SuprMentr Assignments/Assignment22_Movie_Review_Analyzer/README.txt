Assignment Name: Movie Review Analyzer
Description: Build a simple sentiment analyzer and test on 5 reviews

---

1. Introduction

Sentiment analysis is a Natural Language Processing (NLP) technique used to determine whether a text expresses positive, negative, or neutral feelings.

---

2. Objective

To build a simple sentiment analyzer that classifies movie reviews as positive, negative, or neutral.

---

3. Approach

A rule-based approach is used:

* A list of positive words and negative words is created
* Each review is analyzed by counting these words

---

4. Working

* Convert text to lowercase
* Split text into words
* Count positive and negative words
* Compare counts to determine sentiment

---

5. Test Results

The system was tested on 5 movie reviews:

* Positive reviews were correctly identified
* Negative reviews were detected based on keywords
* Neutral reviews were identified when both counts were equal

---

6. Conclusion

The sentiment analyzer works effectively for simple cases. It demonstrates how text data can be analyzed using basic NLP techniques.
