Assignment Name: Image Filter Lab
Description: Use OpenCV to apply grayscale, blur, and edge detection

---

1. Introduction

Image processing is a key part of computer vision. It involves transforming images to extract useful information or improve quality.

---

2. Objective

To apply basic image filters such as grayscale, blur, and edge detection using OpenCV and compare results.

---

3. Filters Used

A. Grayscale Conversion

* Converts a color image into black and white
* Removes color information and keeps intensity

B. Blur (Gaussian Blur)

* Smooths the image
* Reduces noise and details
* Useful before edge detection

C. Edge Detection (Canny)

* Detects boundaries of objects
* Highlights edges in the image

---

4. Working

5. Load image using OpenCV

6. Apply grayscale conversion

7. Apply blur filter

8. Apply edge detection

9. Display original and processed images

---

5. Output

* Original Image → Colored image
* Grayscale → Black and white version
* Blur → Smooth image with reduced noise
* Edges → Highlighted boundaries

---

6. Conclusion

Image filters help in preprocessing and feature extraction. These techniques are widely used in applications like object detection, facial recognition, and medical imaging.
